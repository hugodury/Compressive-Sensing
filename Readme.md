# CompressiveSensing (organisation du projet)

Cette arborescence contient les scripts de base pour la découpe d'images en blocs, ainsi que les éléments de traitement, mesures et dictionnaires.

## Dossiers
- `frontend/` : point d'entrée de l'interface
- `backend/` : traitement et utilitaires
- `Data/` : images, dictionnaires, résultats (ex: `Data/Result/...`)

## Lancer
- Point d'entrée Python : `main.py`
